# SwiftUI 實戰：從宣告式思維到可上架 App

用一個完整專案學會現代 iPhone App 開發方式。

這是一本文字主線清楚、以「習慣養成 App」為核心案例的 SwiftUI 中文實戰書。重點不只放在畫面怎麼做出來，也放在狀態、資料流、非同步、持久化、架構、測試與產品整合怎麼一起長成同一個 App。

## 從哪裡開始

- 第一次閱讀，建議先看 [前言](./00-preface.md)。
- 想直接進主線，從 [第 01 章](./chapters/01-declarative-thinking.md) 開始。
- 如果你對 Swift 或 Xcode 還不夠熟，可以先看附錄：
  - [附錄 A Swift 最小基礎](./appendices/appendix-a-swift-basics.md)
  - [附錄 B Xcode 與 SwiftUI 專案基本操作](./appendices/appendix-b-xcode-and-project-basics.md)
  - [附錄 C 本書常用 Swift 與 SwiftUI 速查](./appendices/appendix-c-swift-and-swiftui-cheatsheet.md)

## 這本書會帶你走到哪裡

- 建立 SwiftUI 的宣告式思維
- 做出可操作的列表、表單與導航介面
- 處理狀態、資料流、非同步與本地持久化
- 整理出夠用的架構、Preview、測試與除錯節奏
- 最後把一個教學專案收成更像產品的 App 原型

## 閱讀建議

這本書的正文是主線；附錄是補票上車的地板。最推薦的方式是先照正文往下讀，卡住時再回頭查附錄，而不是一開始就把附錄當成另一本文法書硬啃完。
