# 封底文案

## 封底長文案

如果你正在學 SwiftUI，你可能已經看過不少範例，也能跟著教學把畫面做出來；但只要需求一變、畫面一多、資料開始流動，專案就很容易突然變得很重。這時候真正卡住你的，往往不是語法，而是你開始不知道畫面應該怎麼切、資料該放哪裡、狀態該由誰持有，最後雖然程式能跑，整個 App 卻還是沒有真正長在一起。

這本書想補上的，就是這段最關鍵、也最常被跳過的路。它不把 SwiftUI 寫成一份 API 清單，也不一開始就把讀者推進過重的架構世界，而是用一個完整的「習慣養成 App」主線專案，帶你從宣告式思維、版面組合、狀態與資料流出發，一路走到表單、動畫、非同步、持久化、架構整理、測試、除錯與產品整合。你不只會看見每個功能怎麼做，更會理解它為什麼要這樣設計，才能真的做出一個可維護、可測、可繼續長大的 SwiftUI App。

## 封底短版

從第一個畫面開始，到做出一個真正像樣的 App。  
這本書帶你用現代 SwiftUI 思維，把版面、狀態、資料流、非同步、持久化與產品整合串成同一條清楚的開發路線。

## 適合讀者

- 已有基本 Swift 語法，想真正學會 SwiftUI 的讀者。
- 想從 UIKit 轉向 SwiftUI 的 iOS 開發者。
- 已經會做畫面，但常卡在狀態管理、資料流與專案結構的人。

## 你會帶走的能力

- 理解宣告式 UI 與狀態驅動畫面的核心思維。
- 做出列表、表單、導航、動畫與互動都更穩的介面。
- 知道非同步載入、本地保存與資料生命週期該怎麼安排。
- 建立夠用的架構、Preview、測試與除錯節奏。
- 把作品整理成更接近真實產品的 App 原型。

## 通路簡介版

一本寫給中文讀者的 SwiftUI 實戰書。全書以「習慣養成 App」為主線，從宣告式思維、版面、狀態與資料流，一路帶到非同步、持久化、架構、測試、除錯與產品整合，幫助讀者從「會照著做」走到「能自己做」。
