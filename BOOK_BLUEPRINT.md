# SwiftUI 中文書藍圖

## 推薦書名方向

推薦主標題：

`《SwiftUI 實戰：從宣告式思維到可上架 App》`

你也可以考慮這兩個替代方案：

- `《寫給中文讀者的 SwiftUI》`
- `《SwiftUI 開發現場：介面、狀態、架構與實戰》`

副標題建議：

`用一個完整專案學會現代 iPhone App 開發方式`

## 這本書寫給誰

### 核心讀者

- 已經知道 `let`、`var`、`struct`、`enum`、`async/await` 等基本 Swift 語法的人。
- 想從 UIKit 轉向 SwiftUI 的 iOS 開發者。
- 想做 App，但在狀態管理、資料流與架構上常常卡住的人。

### 不以這群人為主要對象

- 完全沒有程式基礎的讀者。
- 只想快速抄 UI，而不在意背後原理的人。
- 已經在大型團隊中長期維護複雜 SwiftUI 架構的資深工程師。

## 這本書要解決什麼問題

市面上很多 SwiftUI 內容會卡在兩個極端：

- 只有語法切片，讀者學完還是不知道怎麼拼成一個 App。
- 直接丟大型架構，初學者還沒理解 SwiftUI 思維就先被抽象化壓垮。

這本書的核心承諾是：

`讀完後，讀者可以用現代 SwiftUI 思維完成一個中型 App，並知道怎麼讓它維持可讀、可測、可擴充。`

## 差異化設計

### 1. 先教思維，再教 API

SwiftUI 最大門檻不是元件名稱，而是宣告式 UI、狀態驅動更新、資料流單向傳遞這些觀念。

### 2. 用同一個主線專案貫穿

推薦主線專案：

`習慣養成 App`

這個題材的好處是：

- 容易理解，情境貼近日常。
- 可以自然帶出清單、表單、統計、動畫、持久化、通知等主題。
- 難度夠真實，但不會被商業邏輯拖走。

### 3. 每章都回答三件事

- 這個 API 或觀念是為了解決什麼問題？
- 它在主線專案中用在哪裡？
- 初學者最容易誤會什麼？

### 4. 每章都保留「踩坑區」

SwiftUI 很多挫折其實來自誤解，例如：

- 把 View 當成物件在思考。
- 不清楚資料應該由誰持有。
- 在錯的層級做副作用。
- 看到畫面能跑，就以為設計是可維護的。

## 內容結構

### Part 1：建立 SwiftUI 思維

1. 先懂 SwiftUI：宣告式 UI 到底在想什麼
2. 版面配置與畫面組合：從 Stack 到自訂版面
3. 狀態與資料流：讓畫面跟著資料走

### Part 2：做出可操作的介面

4. 清單、導航與資訊流
5. 表單、輸入與資料編輯
6. 元件化：把重複畫面變成可重用系統
7. 動畫、轉場與手勢互動

### Part 3：讓 App 開始有生命

8. 非同步資料與網路請求
9. 本地資料與持久化
10. 架構設計：從單檔案走向可維護專案
11. Preview、測試與開發節奏
12. 效能、除錯與常見陷阱

### Part 4：從作品走向產品

13. 完整專案整合：做出一個像樣的 App
14. 上架前整理、版本演進與下一步

## 每章固定節奏

推薦每章都用這個順序展開：

1. 用一個真實問題開場。
2. 給出本章最小可理解範例。
3. 把範例接回主線專案。
4. 解釋背後的 SwiftUI 思維。
5. 補上常見錯誤與改寫方式。
6. 用練習題讓讀者自己延伸。

## 視覺與編排設計

### 封面方向

- 底色不要太科技藍，也不要走過度工程感。
- 建議用偏米白或淺灰作底，搭配青綠或亮橘做強調色。
- 圖形語言可以借用 SwiftUI 的堆疊感，例如卡片、圓角區塊、層疊面板。

### 版面語言

- 章首頁固定放三塊資訊：`這章你會學到什麼`、`你會完成哪一段功能`、`需要的前置知識`。
- 內文中的提示框固定三種：`觀念提醒`、`常見陷阱`、`延伸實戰`。
- 程式碼說明不要一次解完全部，優先貼近讀者當下看到的片段。

### 文字語氣

- 用中文把抽象概念講成人話。
- 不要只說「這樣寫比較好」，要說清楚「為什麼這樣比較穩」。
- 避免炫技式句子，重點是帶著讀者把事情做成。

## 建議篇幅

如果你想先寫一本到能出版的長度，建議目標是：

- 12 到 14 章
- 每章 18 到 30 頁
- 全書約 280 到 360 頁

這個篇幅足夠完整，也不會因為範圍過大而寫到中途失速。

## 寫作順序建議

不要照章節順序硬寫到底，建議先寫這四個區塊：

1. 前言
2. 第 1 章
3. 第 3 章
4. 第 13 章的專案總覽

原因很簡單：這四個位置會最早暴露出你這本書真正的教學風格。

## 一句定位

`這不是一本只教你把畫面做出來的 SwiftUI 書，而是一本帶你用正確思維把 App 做對的中文實戰書。`
